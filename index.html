Pet Feeder PRO MAX เป็นเครื่องให้อาหารสัตว์อัตโนมัติที่สามารถควบคุมได้จากทุกที่ผ่านเว็บไซต์
โดยใช้บอร์ด Arduino UNO R4 wifi เชื่อมต่อ MQTT Cloud สำหรับรับ–ส่งคำสั่ง และ จัดเก็บข้อมูลผ่าน Firebase Database
อุปกรณ์มี Ultrasonic Sensor ตรวจจับการปรากฏตัวของสัตว์เลี้ยง พร้อมบันทึกเวลาและแจ้งเตือน Real-time บนเว็บ
ผู้ใช้สามารถสั่งให้อาหารทันทีหรือตั้งเวลาได้ โดยการเปิด–ปิดอาหารควบคุมด้วย Servo Motor

Arduino UNO R4 WiFi

The brain of the project,
providing processing power and Wi-Fi connectivity.

Servo Motor MG90s

micro servo motor with metal gear
can rotate approximately 180 degrees.

Ultrasonic Sensor

description

CONNECTIVITY

Arduino UNO R4 wifi เชื่อมต่อ MQTT Cloud สำหรับรับ–ส่งคำสั่ง ส่งไปที่ Web server
เพื่อแสดงผล และ จัดเก็บข้อมูลผ่าน Firebase Database และ มีการดึงข้อมูลจาก Firebase Database
มาที่ Arduino เพื่อการตั้งเวลาให้อาหาร

Input

- คำสั่งจากผู้ใช้ผ่านเว็บไซต์ (Feed / ตั้งเวลาให้อาหาร)
- สัญญาณจาก Ultrasonic Sensor ตรวจจับการเคลื่อนไหวของสัตว์เลี้ยง

OutPut

การทำงานของ Servo Motor เพื่อเปิด–ปิดการให้อาหาร

ข้อมูลบันทึกการให้อาหารและเวลาที่ตรวจพบสัตว์เลี้ยง
แสดงผลบนเว็บไซต์และเก็บใน Firebase
